# Global Intel Directory

Aplicação web para visualizar informações de países utilizando a API REST Countries.

## Funcionalidades

## Barra de Pesquisa
Na barra de pesquisa é possível buscar países digitando o nome, a capital ou a região.  
A lista de países é atualizada automaticamente conforme o usuário digita.

## Ícone de Busca
O ícone de busca representa a funcionalidade de pesquisa e indica visualmente onde o usuário deve digitar para filtrar os países.

## Filtro por Região
No seletor de região é possível escolher um continente para exibir apenas os países pertencentes àquela região.

## Ordenação por População
Ao clicar no ícone SVG de ordenação, os países são reorganizados com base na população, alternando entre ordem crescente e decrescente.

## Cards de Países
Cada card exibe as principais informações de um país, incluindo bandeira, nome, região, capital, moeda e população.

## Layout Responsivo
O layout se adapta a diferentes tamanhos de tela, como computadores e tablets, utilizando grid.
